# LaTeX Template – Đồ án ngành

Template LaTeX dành cho sinh viên thực hiện Đồ án ngành.

## Bắt đầu

### 1. Tải template
- Nhấn **Code** → **Download ZIP** trên GitHub.
- Hoặc clone repository:
  ```bash
  git clone https://github.com/vnscience/template-latex-do-an-nganh.git
  ```

### 2. Mở project

Chọn **một** trong hai cách:

* **Overleaf:** `New Project` → `Upload Project` → chọn file `.zip`.
* **Prism:** Upload trực tiếp file `.zip` (hoặc thư mục project) để bắt đầu.

### 3. Biên dịch

Mở `main.tex` và **Compile**.

> 📖 Mọi hướng dẫn sử dụng, cấu trúc thư mục và quy định trình bày đã được tích hợp sẵn trong template.

⭐ Nếu template hữu ích, hãy **Star** repository để nhận các bản cập nhật.
